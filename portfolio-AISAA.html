Project Overview

This project documents my transition from web development into data analytics and quantitative systems design. Starting with a foundation in PHP, JavaScript, and HTML, I expanded into Python-based analytics after completing a module in Analytics Programming.

The focus of this work is on using Python to move beyond application development and into data-driven decision-making, including statistical analysis, algorithmic trading strategies, and the exploration of Large Language Models (LLMs) for intelligent systems. This journey reflects a shift from building software interfaces to designing analytical and automated systems that extract insight and intelligence from data.

Key Features

Python-Based Data Analytics

Applied Pandas and NumPy for data cleaning, transformation, and exploratory analysis, forming the foundation for analytical and quantitative workflows.

Quantitative & Statistical Modeling

Designed and evaluated algorithmic trading strategies using statistical methods and backtesting to assess performance and risk.

Intelligent Systems Exploration (LLMs)

Explored Large Language Models and their potential use in analytical systems, automation, and decision-support workflows.

Engineering-to-Analytics Skill Bridge

Integrated software engineering principles with analytics and quantitative thinking, enabling scalable, reproducible, and insight-driven solutions.
